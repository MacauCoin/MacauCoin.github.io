Sic Boは、3つのダイスを使用して結果を決定する伝統的な中国の賭博ゲームです。プレーヤーは、3つのダイスの合計値または特定の組み合わせに賭け、彼らの賭けが正しい場合、対応する賞金を獲得します。

Hash Sic Boは、ブロックチェーン技術を利用した伝統的なシックボーゲームの改良版です。スマートコントラクトを利用して、ゲームの安全性、透明性、公正性を確保し、Binance Smart Chain（BSC）で運営されています。

ゲームに参加するには、プレーヤーはBinance Smart Chain（BSC）上で自分の暗号財布を承認し、USDT（BEP20）を使用して賭けをする必要があります。ウォレットを承認するには、プレーヤーは暗号財布に少なくとも0.0005 BNBを用意し、賭けをするには少なくとも10 USDT（BEP20）が必要です。

賭けをすると、プレーヤーは「roll」ボタンをクリックできます。スマートコントラクトは、賭け金額を自動的にプレーヤーのウォレットから差し引き、トランザクションで生成されたHASHを使用して結果を決定します。規則によれば、HASHの最後の3文字は以下のようにダイスの値に対応します。

1 = 7 = c = i = o = u

2 = 8 = d = j = p = v

3 = 9 = e = k = q = w

4 = 0 = f = l = r = x

5 = a = g = m = s = y

6 = b = h = n = t = z

スマートコントラクトは、賭けと結果に基づいてプレイヤーが勝ったかどうかを決定します。プレイヤーが勝った場合、スマートコントラクトは自動的に賞金を承認された暗号財布に配布します。

従来のギャンブルゲームとは異なり、Hash Sic Boではプレイヤーは他人の口座に入金する必要はなく、勝利後に賞金を換金できないリスクはありません。このゲームはブロックチェーン上で動作するため、HASHを操作することは不可能であり、公正性と透明性が保証されています。
